Dimensional Analysis-Decipher the trinity explosion

Introduction

Dimensional analysis, which is widely used in science and engineering, can be used to analyze or check the relationship between several physical quantities. The important principle of dimensional analysis is that the laws of physics must be independent of the units in which they measure physical quantities. For any meaningful equation, the left-hand and right-hand sides must have the same dimensions. Checking compliance with this rule is the most basic step in dimensional analysis, so the equation is dimensionally consistent.

Example

Let's use an example to decipher this method.

During WWII, British government cooperate with US in the Manhattan project. After trinity test explosion, due to security, the bomb's yield remains unknown while the only thing came to people's eyes is a piece of film recording the bomb. G. I. Taylor studied the movie of the Trinity test explosion, estimating the yield of the bomb. The way Taylor used is dimensional analysis.

 

Let's me list some common dimensions of physical quantities, in capital letters:

Back to this trinity problem, dimensions of the physical variables are:

r​ : the radius of the bomb cloud ~ [L]

ρ: density of air ~ [ML−3]

E: energy released by the device ~ [ML2T−2]

t: time at which the bomb cloud reaches r ~ [T]

 

Assuming that:

r=f(ρ,E,t)

Specifically:

r=CρxEytz

where x,y,z​​ are constants.

 

Based on their dimension:

L=[ML−3]x[ML2T−2]y[T]z

Expand both sides and equate corresponding exponents:

L:1=−3x+2yM:0=x+yT:0=−2y+z

Solve this systems and we can get:

x=−1/5,y=1/5,z=2/5

Thus, we could get the relationship between these variables:

r=C(t2Eρ)1/5

we could also use linear regression to solve the constants by rewirte the model and fit it:

r=atb

take the log for both sides:

log(r)=log(a)+blog(t)

By linear regression, we can calculate the parameter for a​​ and b​​​ based on the data recorded, and we can get the value for parameter b≈0.4094​, which is nearly 2/5

# Based on dimensional analysis, we use the data of radius of the shock front and time to estimate the yield of the
# bomb.




import numpy as np
import matplotlib.pyplot as plt
from scipy.optimize import leastsq
from sklearn.linear_model import LinearRegression
import matplotlib.pyplot as plt


# Taylor studied the movie of Trinity test explosion
rho = 1.25
t = np.array(
    [0.1, 0.24, 0.38, 0.52, 0.66, 0.8, 0.94, 1.08, 1.22, 1.36, 1.5, 1.65, 1.79, 1.93, 3.26, 3.53, 3.8, 4.07, 4.34, 4.61,
     15, 25, 34, 53, 62]).reshape(-1, 1)
r = np.array(
    [11.1, 19.9, 25.4, 28.8, 31.9, 34.2, 36.3, 38.9, 41, 42.8, 44.4, 46, 46.9, 48.7, 59, 61.1, 62.9, 64.3, 65.6, 97.3,
     106.5, 130, 145, 175, 185])
ln_t = np.log(t)
ln_r = np.log(r)
reg = LinearRegression()
reg.fit(ln_t, ln_r)
print(reg.coef_)
# The result is very closed to 2/5, which is from the dimensional analysis
c = np.mean(5 * ln_r - 2 * ln_t)
E = rho * np.exp(c) * 1000000 / (4.182e12)  # unit - Joule, 1kt = 4.184e12 J
print(E)

 

 

Thus,

E≈ρec

where c=5log(r)−2log(t)​​​​,

 

Further calculation tells us that E≈8.6418×1013J​​, since 1kt=4.1848×1012​​, the total yield for bomb is 20.65kt​​​.

We actually estimated the bomb's yield without using any complicated physical formula！
