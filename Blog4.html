Common tricks for proving in probability

Introduction

There are some common tricks if we need to prove especially the case requiring the usage of three main axioms: A1:P(E)≥ for any event E; A2:P(Ω)=1; A3:P(E1∨E2∨..)=P(E1)+P(E2)+... if E1, E2,.. are such that Ei∧Ej=∅ for i≠j.

 

Examples

Suppose that Ω is a sample space on which are defined a probability measure P and events: E, F, and G. Using the above three axioms of probabilty to prove:

i)P(E∧F)≥P(E)+P(F)−1
ii)ifE⊆F,then P(E)≤P(F)
iii)P(E∨F∨G)=P(E)+P(F)+P(G)−P(E∧F)−P(E∧G)−P(F∧G)+P(E∧F∧G)

 

iv)P(E∨F∨G)≤P(E)+P(F)+P(G)

Proof:

i):

P(E∨F)=P(E)+P(F)−P(E∧F)

By A1:

P(E∨F)C≥0 ,and 1−P(E∨F)C≥0
P(E∨F)C=1−P(E∨F)=1−P(E)−P(F)+P(E∧F)≥0

Thus,

P(E∧F)≥P(E)+P(F)−1

ii):

Since E⊆F, we can rewrite F as:

F=(F∧E)∨(F∧EC)P(F)=P(E)+P(F∧EC)

We know that:

 P(F)≥0,P(E)≥0

Thus,

P(E)≤P(F)

iii):

we denote F∨G=H

P(E∨F∨G)=P(E)+P(H)−P(E∧H)=P(E)+P(F)+P(G)−P(F∧G)−P(E∧(F∨G))=P(E)+P(F)+P(G)−P(F∧G)−P(E∧F)−P(E∧G)+P(E∧F∧E∧G)=P(E)+P(F)+P(G)−P(E∧F)−P(E∧G)−P(F∧G)+P(E∧F∧G)

iv):

we denote F∨G=H again:

Since:

P(E∨F)=P(E)+P(F)−P(E∧F)

we know that: P(E∧F)>0, so −P(E∧F)<0

Therefore:

P(E∨F)≤P(E)+P(F)

Back to the LHS of of the inequality:

P(E∨F∨G)=P(E∨H)≤P(E)+P(H)
P(H)=P(F∨G)≤P(F)+P(G)

Thus,

P(E∨F∨G)≤P(E)+P(F)+P(G)

Generalize this property, we could get the Boole's inequality:

for a countable set of events, A1,A2,...

P(⋃iAi)≤∑iP(Ai)

Literally speaking, Boole's inequality tells us that the probability of all events is not greater than the sum of the probabilities of a single event.
