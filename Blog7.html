Grey model for forecasting with Python

Introduction

System theory regards the world as a system composed of interacting parts, which can be determined by relational networks, which provides us a new approach to solve problems from a systematical thinking view. Grey system theory holds the idea that although the behavior phenomenon of the system is hazy and the data is complex, it is orderly and has an overall function. Perhaps it is inspired by Aristotle's belief that knowledge is not obtained by examining individual components, but from an understanding of the whole [1]. Gray forecasting is to generate and process the original data by identifying the development trend of different degrees between system factors and then explore how system changes. Before establishing the grey model, it is necessary to process the data of the original time series, and the data after data preprocessing is called the generated sequence. Among many grey models, GM(1,1) model is most commonly used.

Methods

Let the original data x(0)=(x(0)(1),x(0)(2),...,x(0)(n))

  • Step 1: accumulate the original data to weaken the volatility.
x(1)=(x(1)(1),x(1)(2),...,x(1)(n))

Where

x(0)(t)=∑t=1nx(0)(t), t=1,2,...,n
  • Step 2: Build a first order ODE:
dx(1)dt+ax(1)=b

a and b are constants. Let's construct a matrix β=(a,b)T, if we can get the parameters β, we could get the predicted value for x(0).

  • Step 3: make the average value for the accumlated sequence:
B=[0.5(x(1)(1)+x(1)(2))0.5(x(1)(2)+x(1)(3))..0.5(x(1)(n−1))+x(1)(n))],Yn=(x(0)(2),x(0)(3),...,x(0)(n))T

 

  • Step 4: We use least square method to solve for β
β=(BTB)−1BYn
  • Step 5: Once the parameters are known, we can solve for the ODE
x^(1)(t+1)=(x(0)(1)−ba)exp(−at)+ba
  • Step 6: Get the original sequence
x^(0)(t+1)=x^(1)(t+1)−x^(1)(t)
  • Step 7: Once the model is selected, it must be tested to determine whether it is reasonable. We can calculate the residual.
ϵ(0)(t)=x(0)−x^(0)(t)x(0)(t)

The quality of the model is very high if |ϵ(0)(t)|<0.1. Actually, there are lots of model validation methods. For example, Posterior-variance-test, which is to test the statistical characteristics of the distribution of absolute residuals.

Posterior-variance-test [2]

  1. Average value of the original sequence:
x―(0)=1n∑t=1nx(0)(t)
  1. Mean square deviation of the original sequence:
S1=(∑t=1n(x(0)(t)−x―(0))2n−1)1/2)
  1. Mean of residuals:
Δ(t)=|x(0)(t)−x^(0)(t)|, Δ―=1n∑t=1nΔ(0)(t)
  1. Mean square deviation of residuals:
S2=(∑t=1n(Δ(0)(t)−Δ―(0))2n−1)1/2)
  1. Variance ratio:
C=S2S1

 

  1. Mall residual probability:
P=p{|Δ(t)−Δ―|<0.6745×S1}
  1. Test table: the ranks of forecase precision:

Keep in mind that only the model that passes the test can be used for prediction.

  • Step 8: prediction by our model
x^(0)=[x(0)(1),x(0)(2),...,x(0)(n),x(0)(n+1),...,x(0)(n+m)]

Example

Profits of a company [3] from 1999-2008 are as follows: [89677, 99215, 109655, 120333, 135823, 159878, 182321, 209407, 246619, 300670], now we need to predict the profit for this company in the future.

We use python to develop our grey model for predicting.

import matplotlib.pyplot
import numpy as np
import matplotlib.pyplot as plt
import math as mt


# original sequence
X0 = [89677, 99215, 109655, 120333, 135823, 159878, 182321, 209407, 246619, 300670]


# cumsum
X1 = [89677]
add = X0[0] + X0[1]
X1.append(add)
i = 2
while i < len(X0):
    add = add + X0[i]
    X1.append(add)
    i += 1
print("X1", X1)


M = []
j = 1
while j < len(X1):
    num = (X1[j] + X1[j - 1]) / 2
    M.append(num)
    j = j + 1
print("M", M)


# least square method
Y = []
x_i = 0
while x_i < len(X0) - 1:
    x_i += 1
    Y.append(X0[x_i])
Y = np.mat(Y).T
Y.reshape(-1, 1)
print("Y", Y)


B = []
b = 0
while b < len(M):
    B.append(-M[b])
    b += 1
print("B:", B)
B = np.mat(B)
B.reshape(-1, 1)
B = B.T
c = np.ones((len(B), 1))
B = np.hstack((B, c))
print("c", c)
print("b", B)


# parameters
beta = np.linalg.inv(B.T.dot(B)).dot(B.T).dot(Y)
a = beta[0]
b = beta[1]
const = b / a
print(beta)
print(type(beta))


# predict model
F = [X0[0]]
k = 1
while k < len(X0) + 10:
    F.append((X0[0] - const) * mt.exp(-a * k) + const)
    k += 1
print("F", F)


# get the predicted sequence
x_hat = [X0[0]]
g = 1
while g < len(X0) + 10:
    print(g)
    x_hat.append(F[g] - F[g - 1])
    g += 1


X0 = np.array(X0)
x_hat = np.array(x_hat)
print(x_hat)


# set the time sequence
t1 = range(1999, 2009)
t2 = range(1999, 2019)


# visualization
plt.plot(t1, X0, color='r', linestyle="--", label='true')
plt.plot(t2, x_hat, color='b', linestyle="--", label="predict")
plt.legend(loc='upper right')
plt.xlabel('year')
plt.ylabel('Profit')
plt.title('Profit prediction for company by Grey model')
plt.show()

Summary

The advantages of grey model are that:

  • It does not need a large number of samples.
  • The samples do not need regular distribution.
  • The computation is not complicated.
  • The quantitative analysis results and qualitative analysis results will not be inconsistent.
  • The accuracy is very high.
