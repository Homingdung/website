Understanding the interpolation - a comparative view with fitting

The first thing I'd like to emphasize is the difference between fitting and interpolation, which confuses beginners a lot. Fitting, there is no need for the plot to touch the points, for example, linear regression could fit the data by method like least square method while interpolation requires the plot to pass through points given by the data.

Suppose that we want to use polynomial P(x) to interpolate the data. Let's try the first approach.

Let's define the polynomial Pn(x) and the given points (called nodal points) are x0,x1...xn

Pn(x)=a0+a1x+a2x2+...+anxn=∑j=0najxj

We want to find

∑ajxjn=P(xj)=0

since we require the polynomial to pass through the nodal points, which leads to a matrix problem:

[1x0x02...x0n1x1x12...x1n..1xnxn2...xnn][a0a1..an]=[P(x0)P(x1)..P(xn)]

The first matrix on the LHS is called Vandermonde matrix. In order to define the polynomial, we need to find the solution of the (n+1) by (n+1) matrix problem. However, further problems arise when we try to solve the vandermonde matrix.

 

The second approach is to construct basis, like the span of the linear space.

Let

Pn(x)=span {0,x,x2,...,xn}

Think about the idea of linear independence, which could express the linear space, we require our polynomial to satisfiy few conditions:

  • The degree of the polynomial should be ≤n
  • The basis will only keep the same index term while other terms will vanish.

Thanks to the Kronecker delta, we could express the above conditions easily

Lj(xi)=δij={1i=j0i≠j

Now, we could express our polynomial

Pn(x)=f(x0)L0(x)+f(x1)+...+f(xn)Ln(x)=∑j=0nf(xj)Lj(x)

with the condition:

Pn(xi)=∑j=0nf(xj)Lj(xi)=∑j=0nf(xj)δij=f(xi)

Solve this equation we could get:

Lj(x)=Πi=0,i≠j(x−xi)Πi=0,i≠j(xj−xi)

This kind of method is called Lagrange form of polynomial interpolation and the advantage of this method is that we no longer need to solve large linear system. Lagrange interpolation is very convenient in theoretical analysis. However, in calculation, when one interpolation point is added or reduced, all the corresponding basic polynomials need to be recalculated, so the whole formula will change, which is very complicated. In addition, when there are many interpolation points, the number of Lagrange interpolation polynomials may be very high, so it is characterized by numerical instability. That is to say, although a given value is obtained at several known points, there will be a large deviation from the "actual" value. This kind of phenomenon is also known as Runge's phenomenon, which could be solved by piecewise interpolation polynomial.
